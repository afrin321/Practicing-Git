Learning Git

This an attempt to learn and practice git in a simple way.


Step 1: Create a local repository.

I created a simple html file (this). Added some content to it.

Then I run this command from vscode terminal:
# git init


Step 2: Create a remote repository in Github and Push changes.

I then created a remote Github repo and added the file(s) to the remote github repo.

# git add .
# git commit -m "first commit"
# git branch -M main
# git remote add origin https://github.com/afrin321/Practicing-Git.git
# git push -u origin main


Step 3: Pull and Push changes.

Next, I added a readme.md file from the remote github repository. This created a change in the remote and local repository. Now I first pull the change from the remote repo. By this time, I have also made some changes, so now I will push these changes into remote repo.

# git pull origin main
# git add .
# git commit -m "updated html file"
# git push origin main
